Fetches a block by its blockheight height. Returns the BlockId, BlockBody, and contained transactions IoTransactions of the fetched block, if it exists.

Implementation

@override
Future<(BlockId, BlockBody, List<IoTransaction>)?> blockByHeight(Int64 height) async {
  final req = FetchBlockIdAtHeightReq(height: height);
  final blockId = (await client.fetchBlockIdAtHeight(req)).blockId;

  final response = await blockById(blockId);
  return response;
}
