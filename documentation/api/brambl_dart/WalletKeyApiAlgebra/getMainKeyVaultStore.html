Return the VaultStore for the Topl Main Secret Key.

name - The name identifier of the VaultStore. This is used to manage multiple wallet identities. Most commonly, only one wallet identity will be used. It is the responsibility of the dApp to manage the names of the wallet identities if multiple will be used.

Returns the VaultStore for the Topl Main Secret Key if it exists. If retrieving fails due to an underlying cause, return a WalletKeyException.
